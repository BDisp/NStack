Class Unicode.Script

Static class containing the Unicode script tables.

Inheritance
System.Object
Unicode.Script
Namespace: NStack
Assembly: NStack.dll
Syntax
public static class Unicode.Script
Remarks

There are static properties that can be used to fetch a specific category, or you can use the NStack.Unicode.Script.Get method in this class to retrieve the range table by its script name
